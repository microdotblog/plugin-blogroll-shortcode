# Recommendations blogroll shortcode

Hugo shortcode to show Micro.blog recommendations. Include `{{< blogroll >}}` in a blog post or standalone page.

If you want more control over what is included in the recommendations list, here are some additional variations of using the shortcode:

`{{< blogroll >}}` — show the default "Recommendations" blogroll  
`{{< blogroll name="Science Blogs" >}}` — reference a different blogroll by name  
`{{< blogroll show_hostname=true >}}` — show the blog's hostname in the list  
`{{< blogroll show_about=true >}}` — show the description text you added for a blog
